# tintin-maps

Unicode maps for Jedimud generated with TinTin++.

![](midgaard.png)

## Legend

| Symbol              | Description   |
| ------------------- | ------------- |
| `[ ]`               |  Basic Path   |
| `[$]`               |  Bank         |
| `[M]`               |  Guild Master |
| `[*]`               |  Recall/Login |
| `[&]`               |  Innkeeper    |
| `[D]`               |  Donation     |
| `[#]`               |  Shop/Vendor  |
| `[%]`               |  Fountain     |
| `[ ]+`              |  Exit up      |
| `[ ]-`              |  Exit down    |
| `↖ ↑ ↗ ← · → ↙ ↓ ↘` | One-way path  |


## Usage

Start TinTin++ from terminal

`$ tt++`

Then create a session using the `jedimud.tin` file

`#session j jedimud.com 4000 jedimud.tin`
